## Key Features

<CardGroup cols={3}>
  <Card title="Edge AI Optimized" icon="microchip">
    Purpose-built for embedded devices and edge computing with hot-swappable backends
  </Card>
  <Card title="Multi-Agent Systems" icon="users">
    Native support for AI agents and multi-agent systems with coordination mechanisms
  </Card>
  <Card title="Reinforcement Learning" icon="brain">
    Comprehensive RL algorithms including DQN, PPO, SAC, A2C, and DDPG
  </Card>
  <Card title="Distributed Training" icon="network-wired">
    Scale with MPI, Horovod, and DeepSpeed for large-scale model training
  </Card>
  <Card title="Hardware Acceleration" icon="bolt">
    Multi-backend support: CUDA, ROCm, Vulkan, ARM, NPU, TPU, and more
  </Card>
  <Card title="Quantum Computing" icon="atom">
    Integration with Qiskit and Cirq for hybrid quantum-classical computing
  </Card>
</CardGroup>
